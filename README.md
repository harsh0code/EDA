#  Exploratory Data Analysis (EDA) Project

##  Overview

This project performs **Exploratory Data Analysis (EDA)** on the dataset `trending_videos.csv`.
The goal is to understand the structure, patterns, relationships, and anomalies in the data using statistical and visual techniques.

---

##  Objectives

* Understand dataset structure and features
* Handle missing values
* Analyze distributions of variables
* Detect and visualize outliers
* Identify relationships between variables
* Generate insights using correlation and heatmaps

---

##  Tools & Technologies

* Python 🐍
* Pandas (Data Handling)
* NumPy (Numerical Operations)
* Matplotlib & Seaborn (Visualization)
* Jupyter Notebook / VS Code

---

## Dataset

* File: `trending_videos.csv`
* Contains data related to trending videos (views, likes, comments, etc.)

---

##  Steps Performed

### 1. Data Loading

* Imported dataset using Pandas
* Displayed initial rows and structure

### 2. Data Understanding

* Checked shape, columns, and data types
* Generated summary statistics

### 3. Missing Value Analysis

* Identified missing values
* Visualized using heatmap

### 4. Univariate Analysis

* Histogram plots for numerical features
* Count plots for categorical features

### 5. Outlier Detection

* Boxplots used for visualization
* IQR method used for detection

### 6. Correlation Analysis

* Computed correlation matrix
* Visualized using heatmap

### 7. Bivariate Analysis

* Scatter plots
* Pairplots to understand relationships

---

##  Key Visualizations

* Histograms (Distribution)
* Boxplots (Outliers)
* Heatmap (Correlation)
* Pairplot (Feature Relationships)


---

##  Key Insights

* Some features show strong correlation
* Outliers detected in numerical columns
* Data distribution varies across features
* Certain variables significantly impact trends

---

##  Challenges

* Handling missing values
* Managing outliers
* Visualizing large datasets

---

## Conclusion

EDA helped in:

* Understanding the dataset clearly
* Identifying important patterns and relationships
* Preparing the data for further analysis or machine learning
